# Control de Gastos

App personal para trackear gastos de tarjetas de crédito.

## Stack

- **Frontend**: Streamlit
- **Backend**: Supabase (PostgreSQL)
- **Scraper**: Python + Playwright

## Ejecutar Local

```bash
cd dashboard
streamlit run app.py
```

## Estructura

```
├── dashboard/          # App Streamlit
├── scraper/           # Scrapers de bancos
└── database/          # Schemas SQL
```
